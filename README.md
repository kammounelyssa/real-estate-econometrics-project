# Real Estate & Wildfire Risk Analysis

Academic project conducted at Paris Nanterre University.

## Objective
Analyze the relationship between wildfire exposure, forest proximity and real estate prices in France using spatial and econometric methods.

## Methodology
- Data cleaning
- Spatial matching
- Distance calculations
- Geographic analysis
- Statistical modeling

## Data Sources
- DVF real estate transactions
- Wildfire datasets
- Forest land cover data

## Tools
- R
- QGIS
- SQL
- Excel

## Outputs
- Risk maps
- Distance analysis
- Data visualizations
- Statistical results
